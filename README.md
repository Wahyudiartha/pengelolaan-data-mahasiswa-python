# Pengelolaan Data Mahasiswa Python

Program pengelolaan data mahasiswa sederhana menggunakan bahasa Python.
Program ini memungkinkan pengguna untuk menambah, menampilkan, mencari,
dan menghapus data mahasiswa dengan penyimpanan berbasis file teks (`.txt`).

---

## Fitur Program
- Menambah data mahasiswa (NIM, Nama, Program Studi)
- Menampilkan seluruh data mahasiswa
- Mencari data mahasiswa berdasarkan NIM
- Menghapus data mahasiswa berdasarkan NIM
- Penyimpanan data menggunakan file teks
- Menu interaktif berbasis terminal

---

## Teknologi yang Digunakan
- Bahasa Pemrograman: **Python**
- Penyimpanan Data: **File teks (.txt)**
